Memory Isolation Tester

Register two separate clients, chat as each one, and verify that stored memory stays isolated. This UI talks directly to your current app endpoints: /register and /chat.

Quick Test
  1. Register client A and client B with different emails.
  2. Ask client A to save a fact like your name and preference.
  3. Ask client B what it knows about you.
  4. Only client A should recall the fact.
Expected result Separate tokens should produce separate `user_id` and `team_id` values, so the memory lookup stays isolated.
